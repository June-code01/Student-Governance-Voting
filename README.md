# Student Governance Voting System

Blockchain-based transparent voting platform for student government, clubs, and classroom decisions.

## Overview

A decentralized voting system that ensures transparency, immutability, and fairness in student governance. All votes are recorded on the blockchain, creating an auditable trail of democratic participation.

## Features

- Voter registration with student ID verification
- Proposal creation and management
- Transparent vote counting
- Time-bound voting periods
- Immutable voting records
- Vote history tracking

## Smart Contract Functions

### Public Functions
- `register-voter` - Register as a verified voter
- `create-proposal` - Create a new proposal for voting
- `cast-vote` - Cast your vote on an active proposal

### Read-only Functions
- `get-proposal` - Retrieve proposal details and vote counts
- `get-voter` - View voter information
- `has-voted` - Check if a voter has voted on a proposal
- `get-vote` - Retrieve a specific vote record
- `get-total-proposals` - Get total number of proposals
- `is-proposal-active` - Check if proposal is currently active

## Getting Started

1. Register as a voter with your student ID
2. Browse active proposals
3. Cast your vote on proposals that matter to you
4. View results transparently on the blockchain
